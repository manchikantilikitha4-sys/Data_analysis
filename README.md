DATA ANALYSIS ON CSV FILES 

 Objective
The objective of this project is to analyze sales data using Python and Pandas library. The project helps to understand basic data analysis and visualization techniques.

 Tools Used
- Python
- Pandas
- Matplotlib
- VS Code / Jupyter Notebook

 Description
In this project, a CSV file containing sales data is loaded using Pandas. The data is analyzed to find total sales, average sales, and overall sales performance. The groupby() function is used to calculate total sales for each product. A bar chart is created using Matplotlib to visualize the sales data.

Files Included
- task5_data_analysis.py – Python code for data analysis
- sales.csv – Dataset file
- README.md – Project description

 Outcome
This project helps in understanding how to read CSV files, analyze data using Pandas, and create visualizations using Matplotlib.

 Conclusion
The project successfully analyzes sales data and displays results in both text and graphical format. It improves basic knowledge of data analysis using Python.
